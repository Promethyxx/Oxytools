# Oxytools Laboratory

## Oxy_Watch

> 🚧 Rust module — under construction

- Replicates the Jellyfin workflow to track watched movies
  1. Checks VLC playback status (>90% = watched)
  2. Writes the result to a text file for later tagging
- Requirements: VLC

Oxy_FL
  - There are 3 scripts
  - oxyflf allows you to list folder into a text file
  - oxyfl allows you to list files from the folder and subfolder you design into multiple text files (one by target)
  - OXY_FL_Run is the vbs script to run these script silently
  - Update the path
  - Lets run !

Oxy_JXL
  - There are 3 scripts Powershell
  - Oxyj converts in a lossless mode
  - Oxyjf force the conversion
  - Oxyjp converts into png before JXL
  - You must have cjxl on your pc
  - Fill in the following fields
  - Here is the folder you treat > $SRC = ""
  - Here is the folder where is CJXL > $CJXL = ""
  - Let's run !

Oxy_Wall
  - Python Script to get some 1080 and 2160 wallpaper
  - You need Pexels, Pixabay and Unsplash Api Key in a .env next to the script
  - Requirments are "python-dotenv", "requests"
  - Let's run !
